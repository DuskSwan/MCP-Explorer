# MCP-Explorer

本项目意在学习创建一个MCP客户端，要实现的内容包括

- 通过API调用远端大模型
- 本地部署大模型
- 建立MCP客户端
- 创建一些本地MCP服务器供使用

## 相关库

dotenv: 加载.env文件以获取API（环境变量）

google-genai: 使用 Python 3.9 及更高版本时，请使用以下 pip 命令安装 google-genai 软件包 
pip install -q -U google-genai
